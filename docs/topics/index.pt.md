# Tópicos

Navegue pelo conteúdo do WikiCode por tópico. Os tópicos são intencionalmente amplos para
agrupar muitas páginas; para uma categorização mais detalhada, consulte
[Tags](../tags.md).

## Como os tópicos funcionam

- Cada página pode declarar uma ou mais tags em seu frontmatter.
- Os tópicos são categorias estáveis e de alto nível (`backend`, `frontend`,
  `devops`, `security`, `data`, etc.).
- Tags são rótulos detalhados (`language-go`, `guide`,
  `architecture`).
- Uma página pode ter vários de cada; não há regra de exclusividade.
