# Topics

Browse WikiCode content by topic. Topics are intentionally broad so
they can group many pages; for finer-grained categorization, see
[Tags](../tags.md).

## How topics work

- Each page can declare one or more tags in its frontmatter.
- Topics are stable, high-level categories (`backend`, `frontend`,
  `devops`, `security`, `data`, etc.).
- Tags are fine-grained labels (`language-go`, `guide`,
  `architecture`).
- A page can have several of each; there is no exclusivity rule.
