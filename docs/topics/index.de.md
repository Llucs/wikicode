# Themen

Durchsuchen Sie WikiCode-Inhalte nach Thema. Themen sind bewusst breit gefasst, um viele Seiten gruppieren zu können; für eine feinere Kategorisierung siehe [Tags](../tags.md).

## Wie Themen funktionieren

- Jede Seite kann ein oder mehrere Tags in ihrem Frontmatter deklarieren.
- Themen sind stabile, übergeordnete Kategorien (`backend`, `frontend`,
  `devops`, `security`, `data`, etc.).
- Tags sind feinkörnige Bezeichnungen (`language-go`, `guide`,
  `architecture`).
- Eine Seite kann mehrere von beiden haben; es gibt keine Exklusivitätsregel.
