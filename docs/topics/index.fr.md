# Sujets

Parcourez le contenu de WikiCode par sujet. Les sujets sont volontairement larges pour pouvoir regrouper plusieurs pages ; pour une catégorisation plus fine, consultez [Tags](../tags.md).

## Fonctionnement des sujets

- Chaque page peut déclarer un ou plusieurs tags dans son frontmatter.
- Les sujets sont des catégories stables et de haut niveau (`backend`, `frontend`, `devops`, `security`, `data`, etc.).
- Les tags sont des étiquettes fines (`language-go`, `guide`, `architecture`).
- Une page peut en avoir plusieurs de chaque ; il n'y a pas de règle d'exclusivité.
