# Temas

Explora el contenido de WikiCode por tema. Los temas son intencionalmente amplios para poder agrupar muchas páginas; para una categorización más detallada, consulta [Tags](../tags.md).

## Cómo funcionan los temas

- Cada página puede declarar una o más etiquetas en su frontmatter.
- Los temas son categorías estables y de alto nivel (`backend`, `frontend`, `devops`, `security`, `data`, etc.).
- Las etiquetas son etiquetas detalladas (`language-go`, `guide`, `architecture`).
- Una página puede tener varias de cada una; no hay regla de exclusividad.
