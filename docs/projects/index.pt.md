## Convenções

- Uma pasta por projeto.
- Nome da pasta: minúsculo, com hífen.
- `README.md` explica como instalar, executar e testar o projeto.
- `index.md` é uma visão geral curta e amigável para o site que fornece links para o
  README.
- O código-fonte reside dentro da pasta do projeto, não na raiz do repositório.
