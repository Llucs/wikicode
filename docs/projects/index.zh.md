## 约定

- 每个项目一个文件夹。
- 文件夹名称：小写，连字符分隔。
- `README.md` 说明如何安装、运行和测试项目。
- `index.md` 是简短、面向网站的概述，链接到 README。
- 源代码位于项目文件夹内，而不是仓库根目录。
