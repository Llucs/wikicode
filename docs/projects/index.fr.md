# Projets

Projets réels et exécutables. Chaque projet se trouve dans son propre dossier sous
`projects/`, comprend un `README.md` (pour les développeurs) et un
`index.md` (résumé pour le site), ainsi que le code source fonctionnel.

## Conventions

- Un dossier par projet.
- Nom du dossier : minuscules, avec traits d'union.
- Le `README.md` explique comment installer, exécuter et tester le projet.
- Le `index.md` est un aperçu court et adapté au site qui renvoie vers le
  `README`.
- Le code source se trouve dans le dossier du projet, pas à la racine du dépôt.
