# Projekte

Echte, ausführbare Projekte. Jedes Projekt befindet sich in einem eigenen Ordner unter
`projects/`, enthält eine `README.md` (für Entwickler) und eine
`index.md` (Zusammenfassung für die Website) sowie funktionierenden Quellcode.

## Konventionen

- Ein Ordner pro Projekt.
- Ordnername: Kleinbuchstaben, mit Bindestrich.
- `README.md` erklärt, wie das Projekt installiert, ausgeführt und getestet wird.
- `index.md` ist eine kurze, site-freundliche Übersicht, die auf das README verweist.
- Der Quellcode befindet sich im Projektordner, nicht im Repository-Stammverzeichnis.
