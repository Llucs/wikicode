# Proyectos

Proyectos reales y ejecutables. Cada proyecto se encuentra en su propia carpeta dentro de
`projects/`, incluye un `README.md` (orientado al desarrollador) y un
`index.md` (resumen para el sitio), además del código fuente funcional.

## Convenciones

- Una carpeta por proyecto.
- Nombre de carpeta: minúsculas, con guiones.
- `README.md` explica cómo instalar, ejecutar y probar el proyecto.
- `index.md` es una breve descripción amigable para el sitio que enlaza al
  README.
- El código fuente se encuentra dentro de la carpeta del proyecto, no en la raíz del repositorio.
