# ナビゲーション監査

## 現在の構造

メインナビゲーションは `mkdocs.yml` で定義されており、現在は以下が含まれます。

- ホーム
- 検索
- はじめに
- ラーニングパス
- ガイド
- リファレンス（用語集, アーキテクチャ, 変更履歴）
- トピック
- ツール
- タグ
- ブログ
- プロジェクト
- スニペット
- レポート

## 提案する改善点

1. **メタセクションを「About」または「WikiCode」メニュー項目の下にグループ化**（レポート, 変更履歴, アーキテクチャ）
2. **関連概念であるためタグをトピック内に移動**
3. **最近更新されたコンテンツに視覚的インジケーターを追加**
4. **すべてのナビゲーションレベルでパンくずリストがアクティブであることを確認**

## 実装

すべての変更は `mkdocs.yml` の `nav:` ブロックに対して行う必要があります。
