### 安装

安装过程根据所使用的具体测试库和编程语言而异。以下是以使用Jest的JavaScript项目为例的一般概述：

1. **安装Jest**：
   ```sh
   npm install --save-dev jest
   ```
2. **配置Jest**：在项目目录中添加一个`jest.config.js`文件，包含必要的配置设置。
3. **创建测试文件**：创建一个测试目录结构，通常命名为`__tests__`或`tests`，并使用适当的命名约定添加测试文件（例如`*.test.js`或`*.spec.js`）。

### 基本使用

1. **运行测试**：
   ```sh
   npx jest
   ```
   该命令运行项目中的所有测试文件。

2. **编写一个简单的测试**（以Jest为例）：
   ```javascript
   // example.test.js
   test('add 函数工作正确', () => {
     const add = (a, b) => a + b;
     expect(add(2, 2)).toBe(4);
   });
   ```

3. **运行单个测试**：
   ```sh
   npx jest --testPathPattern 'example.test.js'
   ```

4. **自定义测试路径**：
   ```sh
   npx jest -t "example"
   ```

5. **生成代码覆盖率报告**：
   ```sh
   npx jest --coverage
   ```

此设置为开始使用Jest提供了一个基本框架，但类似的步骤可以适应其他测试库如Mocha或Jasmine。
