# Tauri 開発者ガイド

## Tauri とは？

Tauri は、web テクノロジ (HTML, CSS, JavaScript) を組み合わせたネイティブユーザインターフェース (UI) を作成するためのオープンソース フレームワークです。ブラウザの制約を克服し、web テクノロジを使用してネイティブの体験を提供します。

### キー フEATURES

1. **Web テクノロジ**: 前端に web テクノロジ (HTML, CSS, JavaScript) を使用。
2. **WebAssembly**: WebAssembly を直接アプリケーションで実行して CPU 密集型のタスクをオフロードします。
3. **ネイティブ統合**: ファイル アクセス、クリップボード、システムトray、さらにはそれ以外のネイティブシステム API を提供します。
4. **パフォーマンス**: ネイティブアプリケーションと同等のパフォーマンスを最適化します。
5. **クロスプラットフォーム**: Windows、macOS、Linux で動作します。
6. **ゼロ設定ビルド**: ゼロ設定ビルド システムにより、ビルドプロセスを簡素化します。
7. **カスタマイズ**: プラグインシステムと GTK、Qt などのさまざまな UI フレームワークのサポートにより、高度にカスタマイズできます。
8. **セキュリティ**: セキュリティを考慮した設計で、サンドボックス機能とモジュラーなアーキテクチャを備えています。

## 歴史

Tauri は OpenJS Foundation の OpenJS Foundation Desktop プロジェクトの開発チームによって開発されました。web テクノロジを使用してクロスプラットフォームデスクトップ アプリケーションを効率的でセキュアな方法で構築するためのニーズに対応するために作られました。プロジェクトは大きなトレンドとコミュニティの支援を得て、OpenJS Foundation Desktop プロジェクトから分離し、独立したオープンソース プロジェクトとなりました。

## 使用例

- **プロダクティビティツール**: テキストエディタ、コードエディタ、プロジェクト管理ツールなど。
- **メディアプレイヤー**: 音楽プレイヤー、ビデオプレイヤー、その他のメディア関連のアプリケーション。
- **ユーティリティツール**: ファイルマネージャ、システムモニター、その他のシステムユーティリティ アプリケーション。
- **ゲーム**: ニューゲープーリングや中規模のゲームを含む、ネイティブ体験が必要なゲーム。
- **エンタープライズ アプリケーション**: エンタープライズ使用向けのカスタマイズされたデスクトップ アプリケーション。

## インストール

Tauri を開始するには、システムに Rust と Cargo をインストールする必要があります。以下の手順で Tauri プロジェクトを設定します：

1. **Rust と Cargo のインストール**: フォールドゥックの公式ドキュメントに従って Rust と Cargo をインストールします。
2. **Tauri CLI のインストール**: Tauri CLI を PATH に追加します。
3. **新しい Tauri プロジェクトの作成**:
   ```bash
   cargo tauri init
   ```
   このコマンドは基本的な設定を持つ新しい Tauri プロジェクトを作成します。
4. **ビルドと実行**:
   ```bash
   cargo tauri build
   cargo tauri dev
   ```

## 基本的な使用法

1. **ウェブ アプリケーション**: Tauri アプリケーションの中心は HTML、CSS、JavaScript で構築されたウェブ アプリケーションです。このアプリケーションは Tauri ランタイムによって提供されます。
2. **UI フレームワーク**: Tauri は GTK、Qt、Sycosis などのさまざまな UI フレームワークをサポートします。必要に応じて最も適したものを選択できます。
3. **システム API**: Tauri の提供される API を使用してシステムと相互作用します。例えば、ファイルシステムへのアクセス:
   ```rust
   use tauri::api::fs::{read_dir, read_file, write_file};

   tauri::command!(async fn read_file_command(path: String) -> Result<String, String>) {
       let content = read_file(path).await.map_err(|err| err.to_string())?;
       Ok(content)
   }
   ```
4. **WebAssembly**: ハードウェア負荷を軽減するために WebAssembly モジュールを統合できます。
5. **デプロイ**: Tauri はアプリケーションをさまざまなプラットフォームにパッケージングとデプロイするためのツールを提供します。

## 結論

Tauri は、web テクノロジを使用してネイティブデスクトップアプリケーションを構築するための強力で柔軟なフレームワークです。パフォーマンス、クロスプラットフォームのサポート、豊富な機能の組み合わせにより、効率的でセキュアなデスクトップアプリケーションを構築するための開発者にとって魅力的な選択肢となります。
