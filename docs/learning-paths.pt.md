## Como adicionar uma nova trilha

Adicione uma nova seção a este arquivo com uma lista numerada de links.
Prefira usar páginas existentes. Se uma página ainda não existir, adicione uma tarefa
a [tasks/queue.md](https://github.com/Llucs/wikicode/blob/main/tasks/queue.md)
em vez de inventar conteúdo.
