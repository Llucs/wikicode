# Tags

Todas as tags usadas no WikiCode, geradas automaticamente a partir do frontmatter das páginas. Clique em uma tag para ver todas as páginas que a contêm.

<div id="tag-list" markdown>
!!! info "Lista automática de tags"

    A lista de tags e as páginas associadas a elas é gerada no momento da construção a partir do frontmatter de cada arquivo Markdown em `docs/`, `projects/` e `snippets/`. O tema Material renderiza uma página por tag quando você clica em qualquer tag nos metadados de uma página.
</div>

## Adicionando tags

Adicione tags ao frontmatter de qualquer página:

```yaml
---
title: My page
tags:
  - guide
  - backend
---
```

Prefira reutilizar tags existentes. Se uma nova tag for necessária, adicione-a à lista sugerida em [Guias](guides/index.md#suggested-tags) para que outros autores possam encontrá-la e reutilizá-la.
