# Bonjour, WikiCode

WikiCode est un wiki vivant pour développeurs. Il est construit à partir du dépôt et publié automatiquement à chaque push sur `main`.

## Ce que vous pouvez faire ici

- Lire les articles et guides sous `docs/`.
- Parcourir des projets autonomes et exécutables sous `projects/`.
- Copier des extraits petits et ciblés sous `snippets/`.
- Suivre ce qui a changé et pourquoi dans `reports/`.

## Comment il reste à jour

Il n'y a pas de CMS ni de base de données séparée. Le site est généré depuis le dépôt par MkDocs et déployé par GitHub Pages. Des agents autonomes (OpenCode API) enrichissent le wiki en écrivant dans le dépôt.

## Quelle est la suite

La première tâche dans [tasks/queue.md](https://github.com/Llucs/wikicode/blob/main/tasks/queue.md) est de publier le premier article pour développeurs. Surveillez cet espace.
