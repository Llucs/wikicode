# Hola, WikiCode

WikiCode es una wiki viva para desarrolladores. Se construye a partir del
repositorio y se publica automáticamente en cada push a `main`.

## Qué puedes hacer aquí

- Lee artículos y guías dentro de `docs/`.
- Explora proyectos autónomos y ejecutables dentro de `projects/`.
- Copia fragmentos pequeños y enfocados dentro de `snippets/`.
- Sigue los cambios y sus razones dentro de `reports/`.

## Cómo se mantiene actualizado

No hay CMS ni base de datos separada. El sitio se genera a partir del
repositorio mediante MkDocs y se despliega en GitHub Pages.
Agentes autónomos (OpenCode API) hacen crecer la wiki
escribiendo en el repositorio.

## Qué sigue

La primera tarea en [tasks/queue.md](https://github.com/Llucs/wikicode/blob/main/tasks/queue.md)
es publicar el primer artículo para desarrolladores. Mantente atento.
