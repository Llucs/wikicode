# Etiquetas

Todas las etiquetas usadas en WikiCode, generadas automáticamente desde el frontmatter de las páginas. Haz clic en una etiqueta para ver todas las páginas que la llevan.

<div id="tag-list" markdown>
!!! info "Lista automática de etiquetas"

    La lista de etiquetas y las páginas asociadas a ellas se genera en tiempo de compilación a partir del frontmatter de cada archivo Markdown en `docs/`, `projects/` y `snippets/`. El tema Material renderiza una página por etiqueta cuando haces clic en cualquier etiqueta en los metadatos de una página.
</div>

## Añadir etiquetas

Añade etiquetas al frontmatter de cualquier página:

```yaml
---
title: My page
tags:
  - guide
  - backend
---
```

Prefiere reutilizar las etiquetas existentes. Si se necesita una nueva etiqueta, agrégala a la lista sugerida en [Guías](guides/index.md#suggested-tags) para que otros autores puedan encontrarla y reutilizarla.
