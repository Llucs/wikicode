# Tags

All tags used across WikiCode, generated automatically from page
frontmatter. Click a tag to see all pages that carry it.

<div id="tag-list" markdown>
!!! info "Automatic tag list"

    The list of tags and the pages associated with them is generated
    at build time from the frontmatter of every Markdown file in
    `docs/`, `projects/` and `snippets/`. The Material theme renders
    a per-tag page when you click any tag in a page's metadata.
</div>

## Adding tags

Add tags to the frontmatter of any page:

```yaml
---
title: My page
tags:
  - guide
  - backend
---
```

Prefer reusing existing tags. If a new tag is needed, add it to the
suggested list in [Guides](guides/index.md#suggested-tags) so other
authors can find and reuse it.
