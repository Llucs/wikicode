## Conventions

- Un dossier par extrait (ou par famille d'extraits).
- Nom du dossier : en minuscules, avec traits d'union, souvent préfixé par le langage
  (`python-debounce/`, `go-retry/`).
- Le code proprement dit se trouve dans un seul fichier exécutable.
- `index.md` explique le problème, l'approche et comment adapter l'extrait.
