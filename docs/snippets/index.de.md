# Snippets

Kleine, fokussierte, ausführbare Code-Snippets. Jedes Snippet lebt in
einem eigenen Ordner mit einer kurzen `index.md`, die das Problem
beschreibt, das es löst, und der Datei, die den Code enthält.

## Konventionen

- Ein Ordner pro Snippet (oder pro Snippet-Familie).
- Ordnername: Kleinbuchstaben, mit Bindestrich, oft mit Sprachpräfix
  (`python-debounce/`, `go-retry/`).
- Der eigentliche Code befindet sich in einer einzigen, ausführbaren
  Datei.
- `index.md` erklärt das Problem, den Ansatz und wie das Snippet
  angepasst wird.
