Small, focused, runnable code snippets. Each snippet lives in its own
folder with a short `index.md` describing the problem it solves and
the file containing the code.

## Conventions

- One folder per snippet (or per snippet family).
- Folder name: lowercase, hyphenated, often prefixed by language
  (`python-debounce/`, `go-retry/`).
- The actual code is in a single, runnable file.
- `index.md` explains the problem, the approach and how to adapt the
  snippet.
