## Cómo añadir una nueva ruta

Añade una nueva sección a este archivo con una lista numerada de enlaces.
Prefiere usar páginas existentes. Si una página no existe aún, añade una tarea
a [tasks/queue.md](https://github.com/Llucs/wikicode/blob/main/tasks/queue.md) en lugar de inventar contenido.
