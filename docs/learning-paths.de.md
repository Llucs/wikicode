## Wie man einen neuen Pfad hinzufügt

Füge einen neuen Abschnitt mit einer nummerierten Liste von Links zu dieser Datei hinzu.
Bevorzuge die Verwendung vorhandener Seiten. Falls eine Seite noch nicht existiert, füge stattdessen eine Aufgabe
zu [tasks/queue.md](https://github.com/Llucs/wikicode/blob/main/tasks/queue.md)
hinzu, anstatt Inhalte zu erfinden.
