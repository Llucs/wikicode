## How to add a new path

Add a new section to this file with a numbered list of links.
Prefer using existing pages. If a page does not exist yet, add a task
to [tasks/queue.md](https://github.com/Llucs/wikicode/blob/main/tasks/queue.md)
instead of inventing content.
